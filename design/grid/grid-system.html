小学生の頃、新聞を作る課題ってありませんでしたでしょうか。私はありました。それを思い出して…

CSSのグリッドレイアウトを用いて新聞風なデザインをしてみました。

CSSのグリッドレイアウトとは、

display: grid;
grid-template:
"... ..... ..... ..... ..... ..... ..... ..."
"... header header header header header header ..." 30px
"... center2 center1 center1 center1 center1 title ..." 300px
"... center2 center1 center1 center1 center1 title ..." 100px
"... center2 center1 center1 center1 center1 title-sub ..." 150px
"... center3 center3 center3 center3 center3 center3 ..." 300px
"... center4 center4 center7 center5 center5 center5 ..." 150px
"... center4 center4 center6 center6 center6 center6 ..." 150px
"... add1 add2 add2 add3 add4 add4 ..." 300px
" ...footer footer footer footer footer footer ..." 30px
"... ..... ..... ..... ..... ..... ..... ..." 30px
/ auto 300px 300px 300px minmax(300px, 2000px) 100px 100px auto;

上記のような可読性の高いコードをCSSに記載するだけでレイアウトが組める機能です。
従来のFlexBoxなどと比べタグの数を大幅に減らせます。
また以下の例では行っておりませんが、容易にレスポンシブデザインにすることも可能とのことです。
InternetExploreを除くほとんどブラウザに対応しています。

今月のガス代　判明

　2日午前9時、今月のガス代を示す請求明細書が届いた。毎月一定額を保ち続けていたガス代であったが、今月急激な上昇が見られた。関係者への取材によると先月は連日お湯を張っていたことが分かり、「この時期になると湯船に浸かりたくなる。大きな反省点だ」と心境を述べた。

UberEatsのアプリをダウンロード

　関係者らへの取材によると昨日、UberEatsのアプリをダウンロードしたことが判明した。UberEatsとは手数料などを払うことで自宅など指定の場所へ食事を運んでくれるサービスのことで近年話題を呼んでいる。 宇都宮市でも一部地域でサービスが提供されており、新しいサービスに目がない筆者はダウンロードを開始した。しかしながら想像以上の価格の高さから購入を断念し、結果としてアプリだけがスマートフォンに残る形となった。

消耗品の備蓄始まる

　本格的な冬の寒さを前にして、各種消耗品の備蓄が行われた。米Amazon社によるサイバーマンデーとも重なり、比較的安価に商品を購入できたと関係者らは述べている。このことについて自身は「消耗品が無くなっていないかを気にして脳を無駄に使うことが減った」と胸の内を語った。 シャンプーやボディーソープなどを購入し、半年は購入の必要がないと想定されている。

朝御飯　どうする？

　近年話題となっているプチ断食を受け、朝御飯を省くファスティングの実施に踏み切った。これを行なうことにより日中の眠気の防止や専門家らの意見によると朝にバターコーヒーを飲むことがよいとされている。また従来までの1日3食は時代にそぐわないとの意見も散見された。
　朝御飯を摂らないことで、朝食選びをしなくても良いという恩恵が得られている一方で途中空腹に耐えかねて間食をとってしまい、結果として意味がない現状となっていることも事実である。

餅 購入

　昨日スーパーを訪れた際、割引のポップに気を取られてしまい、一人暮らしにも関わらず誤って大容量1kgの餅を購入した。餅は鍋に入れる際に大変有意義ではあるものの、我が家の貯蓄スペーズを圧迫しており、購入者は反省の色を示している。

実家帰省

　今月26日から予定されている実家への帰省について慎重な態度を示していることが4日の記者会見で判明した。東京都によるGoToトラベル自粛要請を受け、帰路に東京都が含まれることから、場合によっては中止も止むをえないとのことである。
